В этом учебном проекте мы разрабатываем модель, которая поможет бизнесу принимать решения только на основе анализа данных и бизнес-моделирования. Модель должна предсказывать вероятность снижения покупательской активности в следующие три месяца.

Задачи проекта
Разработать предиктивную модель для классификации покупательской активности клиентов.
Сегментировать клиентов по различным признакам для улучшения маркетинговых стратегий.

Данные
Для работы были использованы следующие группы признаков:
Коммуникация сотрудников компании с клиентом.
Продуктовое поведение покупателя (какие товары и как часто покупает).
Покупательское поведение (сколько тратит в магазине).
Поведение на сайте (просмотры страниц, время на сайте).

Использовались модели: Decision Tree, K-Nearest Neighbors (KNN), Logistic Regression, Support Vector Machines (SVM).
Настройка гиперпараметров через GridSearchCV.
Лучшая модель — SVC с ROC-AUC = 0.92.
